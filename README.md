# 🚀 Basic Windows Optimization Pack

---

A curated collection of essential tools and scripts designed to streamline and enhance your Windows operating system. Whether you aim to improve performance, maintain a cleaner system, or better manage resources, this pack provides the building blocks to achieve that efficiently.

---

## 📁 Project Structure

- **browsers/**  
  Contains installers for [Thorium](https://thorium.rocks/), [Brave](https://brave.com/), and [Firefox](https://www.mozilla.org/firefox/) browsers.  
  *Note: No additional optimizations included.*

- **nsudo/**  
  Includes the powerful [NSudo](https://github.com/M2Team/NSudo) utility, allowing execution of processes with elevated system privileges (System, TrustedInstaller) for advanced administrative control.

- **services/**  
  Scripts designed to disable unnecessary Windows services, helping to optimize resource usage and system responsiveness.

- **tweaks/**  
  Contains third-party optimization tools and scripts, including [FR33THY](https://www.youtube.com/c/FR33THY) and [Chris Titus Tools](https://christitus.com/), providing advanced tweaks to improve performance, stability, and Windows customization.

- **7zip (shortcut)**  
  Shortcut to the popular [7-Zip](https://www.7-zip.org/) file archiver — efficient compression and extraction made simple.

- **Autoruns.exe**  
  [Autoruns](https://docs.microsoft.com/sysinternals/downloads/autoruns) from Microsoft Sysinternals, an advanced startup program manager to help speed up boot times.

- **C++.ps1**  
  PowerShell script for downloading and installing the latest Microsoft Visual C++ Redistributable packages.

- **Cleano.exe**  
  Lightweight system cleaning tool to remove temporary files, caches, and other junk that may slow down your PC.

- **Direct X.ps1**  
  PowerShell script to download and install DirectX, a critical component for multimedia and gaming performance.

- **geek (shortcut)**  
  Shortcut to the official [Geek Uninstaller](https://geekuninstaller.com/), a robust tool for complete program removal.

- **windows.ps1**  
  PowerShell script designed to safely and efficiently activate the Windows operating system.

---

## ⚠️ Important Notes

- Always review scripts before execution to understand their effects.  
- Most optimization tasks require administrator privileges.  
- Create a system restore point or backup before applying major changes.
